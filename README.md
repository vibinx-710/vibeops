# VibeOps - Production MLOps Platform

End-to-end MLOps platform for predictive maintenance using industrial sensor data.

## Stack
- Apache Airflow - Pipeline Orchestration
- MLflow - Experiment Tracking & Model Registry
- FastAPI - Model Serving
- vLLM - LLM Inference
- Prometheus + Grafana - Monitoring
- Kubernetes - Container Orchestration
- Terraform - Infrastructure as Code

## Problem Statement
Predict machine failure before it happens using AI4I 2020 Predictive Maintenance Dataset.
